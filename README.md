# Aplikasi Graf dan Dijkstra
---
Inisialisasi:
```
$ git clone https://github.com/InfinityNichto/projek-akhir2
$ cd projek-akhir2
$ chmod +x ./init.sh
$ ./init.sh
```
Kompilasi:
```
$ cd build
$ ./make-n-run.sh
```
Projek dikompilasi di platform Linux menggunakan CMake v4.0.2 dan kompiler clang v19.1.7.
